# cc-plugins

A collection of Claude Code plugins for productivity and ecosystem monitoring.

## Plugins

| Plugin | Description | Status |
|--------|-------------|--------|
| [cc-upgrade](./cc-upgrade/) | Monitor Anthropic ecosystem changes and get prioritized upgrade recommendations | v0.1.0 |

## Installation

Each plugin can be installed independently:

```bash
git clone https://github.com/tjkang/cc-plugins.git
claude plugin add /path/to/cc-plugins/<plugin-name>
```
